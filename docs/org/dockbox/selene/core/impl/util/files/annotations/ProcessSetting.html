Annotation ProcessSetting

  • All Implemented Interfaces:
    java.lang.annotation.Annotation 
    @Retention(value = RetentionPolicy.RUNTIME)@Target(value = ElementType.FIELD)@Documented() 
public @interface ProcessSetting

    Indicates that, when this annotation is present alongside a ninja.leaping.configurate.objectmapping.Setting annotation, some transformation of the node should occur at the following moments:

    • before the value of the field is set from the ninja.leaping.configurate.ConfigurationNode
    • after the value of the field is read from the ninja.leaping.configurate.ConfigurationNode , but just before it is written to the file

    SettingProcessor s are used to take the ninja.leaping.configurate.ConfigurationNode s and alter them directly before the final value for the task is read.
