Package 

Class ConfigurateManager

  • 
public abstract class ConfigurateManager

    Low-level interface defining functionality to interact with Configurate based file types. While this type encourages the usage of Configurate based instances, it is possible to create implementations for alternative configuration libraries and/or frameworks.
